# Thriskeia Database Management Application

**Course:** Advanced Database Topics (Προηγμένα Θέματα Βάσεων Δεδομένων)

## Description

Web application for managing a religious content database. The application provides full CRUD (Create, Read, Update, Delete) operations for system entities including churches, clergy, believers, donations, baptisms, and catechism classes.

## Features

- **Church Management:** Add, edit, delete, and view churches
- **Clergy Management:** Manage clergy members and their information
- **Believer Management:** Manage believers and their details
- **Donation Management:** Record and manage donations to churches
- **Baptism Management:** Record baptisms by parish
- **Catechism Management:** Manage catechism class schedules

## Tech Stack

- **Backend:** PHP 7.4+
- **Database:** MySQL 5.7+
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Server:** Apache (XAMPP)

## Installation

1. Place all files in the XAMPP `htdocs` folder
2. Start Apache and MySQL from XAMPP Control Panel
3. Import `database.sql` via phpMyAdmin
4. Open `http://localhost/thriskeia_webapp/` in your browser
